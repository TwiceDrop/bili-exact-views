# B站精确播放量

一个 Chrome Manifest V3 扩展：在 B 站视频详情页的播放、弹幕、点赞、投币、收藏和分享简写数量旁显示完整数字，例如 `1.0亿（100,842,324）`、`277.6万（2,776,123）`。

## 安装

1. 在 Chrome 地址栏打开 `chrome://extensions`。
2. 开启右上角的“开发者模式”。
3. 点击“加载已解压的扩展程序”，选择本文件夹。
4. 打开或刷新任一 `https://www.bilibili.com/video/...` 视频详情页。

## 支持范围

- 仅支持 Chrome 和 B 站视频详情页；不处理首页、搜索、收藏和推荐卡片。
- 每次进入新视频或站内切换视频时查询一次；不轮询、无设置页、无本地存储。
- 精确数值来自 B 站公开视频详情接口 `https://api.bilibili.com/x/web-interface/view`。

## 限制

若 B 站更改详情页的 DOM 结构、接口返回错误、网络不可用或触发限流，扩展会静默不显示精确数，不影响页面播放功能。
